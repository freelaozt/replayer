Privacy Policy (隐私政策)

Data Collection: This App (Re-ingPlayer) is a local media player. We DO NOT collect, storage, or transmit any personal information, playback history, or files from your device. (数据收集：本应用是一个本地媒体播放器。我们不收集、存储或传输您的任何个人信息、播放历史或设备文件。)

Network Access: The App requires network access only for playing user-specified network streams (e.g., SMB, WebDAV) and fetching metadata from third-party APIs (e.g., TMDB, Bangumi). This data is processed locally and not sent to our servers. (网络访问：应用仅在播放用户指定的网络流及获取第三方 API 元数据时需要网络权限。这些数据仅在本地处理，不会发送至我们的服务器。)

Third-party Services: We use third-party APIs for metadata. Please refer to their respective privacy policies (TMDB, Bangumi, ASSRT). (第三方服务：我们使用第三方 API 获取元数据。请参阅各服务商的隐私政策。)

Contact: If you have any questions, please contact us via GitHub Issues. (联系方式：如有任何疑问，请通过 GitHub Issues 联系我们。)
